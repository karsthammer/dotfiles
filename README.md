My dotfiles configuration
====================

Clone repository
----------------

```bash
git clone https://github.com/karsthammer/dotfiles.git ~/.dotfiles --recursive
```

Mac setup
----------

```bash
~/.dotfiles/setup-mac.sh
```


VIM setup
------------

```bash
ln -s ~/.dotfiles/vimrc ~/.vimrc
ln -s ~/.dotfiles/gvimrc ~/.gvimrc
```
